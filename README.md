# Projeto-RPG-Simulado

Neste repositório, está alocado um sistema com diversas funcionalidades simuladas a um jogo de RPG, com objetivo de se aprofundar na linguagem PHP, POO, Banco de dados, testes e integração com Front-End.

O mesmo está responsável por trazer o back-end do sistema, na qual existirá outro repositório com a responsabilidade de realizar paginas, layouts e entre outras funcionalidades com React e JavaScript.

<hr>

Projeto pessoal para estudo, e com o principal objetivo de chegar perto de boas práticas usadas no mercado.

# Passos:

Inserir Laravel ao projeto.
Inserir Typescript ao projeto.
Inserir Docker ao projeto.
Inserir RabbitMQ ao projeto.
Inserir AWS Cloud ( EC2 ou S3 ) ao projeto.
